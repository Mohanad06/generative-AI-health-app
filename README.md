# AI Health & Environment Advisor

An AI-powered web application built using AWS PartyRock that generates personalized health and environmental insights based on user input.

## Live Demo
🔗 [https://lnkd.in/dqzdcwQZ](https://partyrock.aws/u/Mohanad06/nRdsU4gjg/Health-and-Environment-Advisor)

## Overview
Users enter:
- Age
- Weight
- Height

The application generates AI-based recommendations using prompt engineering techniques.
